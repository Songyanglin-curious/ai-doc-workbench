# Docs Index

本文件是按输入路由表，不是固定阅读链。

## 硬规则

- 不要把 `docs/index.md` 理解成默认阅读顺序。
- 模糊任务先在这里选目标，不默认进入 `context/`。
- 只有当其他目录都不足以支持定位时，才进入 `context/`。

## 路由表

- 已明确给出文件、函数、报错、页面或目录：
  - 直接去用户指定目标
- 需要整理原始材料：
  - 去 `input/`
- 需要放置或查看协议、模板、规范、第三方约束：
  - 去 `references/`
- 需要判断项目当前真相、模块状态或结构边界：
  - 去 `baseline/`
- 需要推进复杂任务收口：
  - 去 `plans/`
- 需要看历史落地事实：
  - 去 `logs/`
- 以上都不足以支持定位：
  - 去 `context/`
