# References

本目录只记录外部参考与约束。

## 放什么

- 协议
- 导入导出模板
- 第三方接口说明
- 行业规范
- 示例材料
- 供应商文档摘录

## 不放什么

- 项目当前主真相
- 当前全局基线
- 模块边界的最终结论

## 推荐索引字段

| 文件 | 类型 | 来源 | 关联模块 | 备注 |
| --- | --- | --- | --- | --- |
| `<file>` | `<protocol/template/spec/vendor/example>` | `<source>` | `<module-or-none>` | `<note>` |

## 规则

- 由参考材料推导出的项目结论，必须进入 `docs/baseline/`
- 本目录默认先扁平存放，量大后再细分子目录
