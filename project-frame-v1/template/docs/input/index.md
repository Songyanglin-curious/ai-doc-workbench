# Input

本目录只记录原始输入。

## 放什么

- PM 原始需求
- 截图
- 会议记录
- 用户描述
- 外部复制来的原文材料

## 不放什么

- 项目最终结论
- 当前基线
- 模块边界
- 复杂任务计划

## 推荐索引字段

| 文件 | 来源 | 日期 | 关联模块 | 是否已被吸收 |
| --- | --- | --- | --- | --- |
| `<file>` | `<source>` | `<YYYY-MM-DD>` | `<module-or-none>` | `<yes/no>` |

## 规则

- 原始输入被吸收后，结论写入 `docs/baseline/`
- 原件仍留在本目录，不要删除
