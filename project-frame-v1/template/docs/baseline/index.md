# Baseline

本目录承载项目当前真相与长期收敛方向。

它不是资料库，也不是任务轨迹。

## 优先级

1. `principles.md`
2. `core.md`
3. `modules/<模块>/index.md`

## 适用边界

- 判断长期收敛方向：读 `baseline/`
- 判断当前认可的模块状态或结构边界：读 `baseline/`
- 判断当前实现实际如何工作：看 live repo
- 判断本轮如何收口：看 `plans/`
- 判断曾发生过什么：看 `logs/`

## 目录入口

- [principles.md](principles.md)
- [core.md](core.md)
- [modules/index.md](modules/index.md)
