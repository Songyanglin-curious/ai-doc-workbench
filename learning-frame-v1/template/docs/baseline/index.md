# Baseline

`baseline/` 只记录已确认能力真相和长期收敛方向。

它不是资料库，不是学习日记，也不是 AI 的猜测区。

## 优先级

1. [principles.md](principles.md)
2. [learner.md](learner.md)
3. [confirmed.md](confirmed.md)

## 进入条件

只有在下面情况才读或改 `baseline/`：

- 判断长期学习方向是否偏离
- 判断某个能力是否已经正式确认
- 更新个人学习原则
- 把经过无 AI 验证的能力事实写入确认区

## 写入规则

- 必须有人确认。
- 必须有 `evidence/` 证据链接。
- 不写“我感觉懂了”。
- 不写未验证的 AI 总结。
- 不写外部资料摘要，外部资料去 `references/`。
