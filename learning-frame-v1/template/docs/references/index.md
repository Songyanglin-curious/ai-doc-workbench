# References

`references/` 放长期可引用资料。

可以放：

- 教程
- 论文
- 文章
- 官方文档
- 书籍章节索引
- 学习方法资料

规则：

- references 是资料，不是已掌握事实。
- 资料进入 references 后，仍需要通过 `map/` 和 `evidence/` 才能影响 baseline。
- 对事实型、工具型、价格/隐私/政策类资料，要优先保留原始链接和访问日期。

## 推荐来源类型

- 官方文档
- 教材或经典书
- 同行评审论文
- 高质量课程
- 可复现实验或代码

## 警惕来源

- 只讲爽感的经验帖
- 只有结论没有验证的 AI 总结
- 商业软文
- 过度承诺“快速精通”的教程
