# Docs Index

本文件是输入路由表，不是固定阅读链。

## 路由表

- 想知道这套框架怎么用、复制快捷指令：
  - 去 `使用/`
- 现在要学什么、下一步做什么：
  - 去 `current/`
- 想知道某个能力和哪些前置、场景、链路相关：
  - 去 `map/`
- 想确认自己是否真的掌握：
  - 去 `evidence/`
- 想查看已经确认的能力真相和长期原则：
  - 去 `baseline/`
- 本轮原始材料、临时问题、对话记录：
  - 去 `input/`
- 教程、论文、文章、长期参考资料：
  - 去 `references/`

## 升级条件

只有在下面情况才从当前任务升级到框架维护：

- 当前任务暴露了新的前置缺口
- 当前任务改变了主学习路线
- 产生了无 AI 验证结果
- 发现了夹生风险
- 某个能力可以确认进入 baseline
