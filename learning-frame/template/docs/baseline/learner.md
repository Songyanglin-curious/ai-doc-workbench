# Learner Profile

本文件记录个人学习约束，不追求完整，只写会影响学习路线的真实信息。

## 当前定位

- 主要目标：系统补齐工程能力，能在工作中解决真实问题
- 学习方式偏好：低摩擦、内容核心、不要模板负担
- 主要风险：看懂 AI 输出后误以为自己会了
- 特别关注：基础概念清晰、前置关系清楚、避免夹生饭

## 当前重点方向

- 编程基本功与问题建模
- 后端、接口、网络、权限、数据库等工程能力
- AI 辅助学习和开发中的独立判断

## 不希望 AI 替我做

- 直接给完整答案后让我被动看懂
- 替我判定“已经掌握”
- 为了显得系统化而生成大量空文档
- 静默改变学习路线

## 可接受的 AI 帮助

- 诊断薄弱点
- 拆成小台阶
- 解释概念边界
- 生成检索题和迁移题
- 对错误做归因
- 维护 map 关系并汇报变更
