# Prompt Shortcuts

这些不是必须使用的咒语，只是降低启动摩擦的短指令。

## 1. 开始一个学习任务

```text
按 learning-frame 帮我学习下面内容。
先定位它对应的问题场景、能力节点和当前台阶。
不要直接给完整答案，先做诊断，再给最小解释和一个无 AI 验证任务。
内容：
```

## 2. 防止夹生饭

```text
请检查我是否可能夹生。
不要夸我，也不要直接重讲。
请按：概念边界、反例、变式任务、前置缺口 四项检查。
如果发现不稳，请指出最近应该回退的前置台阶。
```

## 3. 生成无 AI 验证

```text
请为这个能力节点生成无 AI 验证任务。
要求：
1. 不需要查资料也能完成
2. 能暴露概念边界
3. 有一个变式/迁移题
4. 给出通过标准，但先不要给答案
能力节点：
```

## 4. 错因分析

```text
下面是我的无 AI 验证结果。
请只做错因分析，不要替我重写答案。
按：概念不清 / 提取失败 / 迁移失败 / 执行粗心 / 前置缺口 分类。
最后给出下一步最小台阶。
```

## 5. 维护 map

```text
请按 learning-frame 维护 map。
你可以建议新增或修改能力节点、前置关系、应用场景和能力链。
但主链、关键前置、掌握状态、baseline 写入必须列为“需要我确认”。
最后输出变更摘要。
```
