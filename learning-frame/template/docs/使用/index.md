# 使用

本目录只回答一个问题：这套学习框架怎么用。

它不是资料区，也不是学习记录区；不要把日常学习内容写到这里。

## 文件说明

- [运行协议.md](运行协议.md)
  - 给 AI 优先看，也给你在卡住、验证、维护时看。
  - 说明快速学习、深挖、验证、维护四种模式。
- [快捷指令.md](快捷指令.md)
  - 给你开新对话或发起学习任务时复制使用。
  - 它不是硬规则，只是降低启动摩擦。

## 使用边界

- 想开始用框架：先看根目录 `START-HERE.md`。
- 想约束 AI 行为：AI 先看根目录 `AGENTS.md`。
- 想知道当前任务怎么跑：看 `运行协议.md`。
- 不知道怎么向 AI 开口：看 `快捷指令.md`。
